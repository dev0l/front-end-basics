HTML Basics

What is HTML?

HTML stands for Hyper Text Markup Language and is the standard markup language for creating Web pages. It describes the structure of a Web page and consists of a series of elements which tell the browser how to display the content. Elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

A HTML file is just a document with hte file extension .html containing HTML code. You can use any editor to create HTML, even notepad but that takes complete knowledge and a lot more work. To make it easier we use an IDE (Integrted Development Environment).

HTML Page Structure

<html>
<head>
<title>Page title</title>
</head>
<body>
<h1>This is a heading</h1>
<p>This is a paragraph.</p>
<p>This is another paragraph.</p>
</body>
</html>

  1. Create a HTML file:

    1. We can create a new file with notepad. Click Save As and name it "index.html" as File name:. Make sure that "All files" is selected as Save as type: and that "UTF-8" is selected as Encoding: then click Save. We now see that the file is a HTML file and if you double click it you will load it in the browser.

    2. We can also create a new html file in our IDE, which is a lot easier. If you are not familiar with "Visual Studio Code", check out the "Get Started With Visual Studio" guide here..

  2. Start a simple HTML document:

    In Visual Studio Code you can simply create a html document by typing html:5 and pressing Tab on the keyboard. This will generate the head and body of the HTML document.

    It will look somethng like this: 
          <!DOCTYPE html> <!-- This declaration is not an HTML tag. It is an "information" to the browser about what document type to expect. -->
          <html lang="en"> <!-- This tag represents the root of an HTML document. It is the container for all other HTML elements. -->
          <head> <!-- This element is a container for metadata (data about data). -->
              <meta charset="UTF-8"> <!-- This tag defines metadata about an HTML document. Metadata is data (information) about data. -->
              <meta name="viewport" content="width=device-width, initial-scale=1.0">
              <title>Document</title> <!-- This element defines the title of the document. The title must be text-only, and it is shown in the browser's title bar or in the page's tab. -->
          </head>
          <body> <!-- This tag defines the document's body. It contains all the contents of an HTML document. -->

          </body>
          </html>

  1. Add your first content to the document:

    First we have to understand what an HTML element is. As seen above, an element is defined by a opening tag (start tag): followed by some content followed by a closing tag (end tag). The closing tag always starts with a forward slash "/".

    <tagname> Content goes here... </tagname>

    The HTML element is everything from the opening (start) tag to the closing (end) tag.

    <tagname> : Opening tag

    </tagname> : Closing tag

    1. HTML Headings and Paragraphs

      HTML headings are defined with the <h1> to <h6> tags. <h1> defines the most important heading. <h6> defines the least important heading.

      HTML paragraphs are defined with the <p> tag.

      As an example, let's create a heading and a paragraph inside the body: 

        <body>

    <h1>My first heading</h1>

    <p>My first paragraph</p>

  </body>

    2. HTML Links

      HTML links are defined with the <a> tag:

      <a href="https://www.w3schools.com">This is a link</a>

      The link's destination is specified in the href attribute.

      Attributes are used to provide additional information about HTML elements. We will learn more about attributes later.

    3. HTML Images

      HTML images are defined with the <img> tag.

      <img src="skaggbyran.jpg" alt="skaggbyran.se" width="104" height="142">

      The source file src, alternative text alt, width, and height are provided as attributes.

  1. Nested HTML Elements:

    HTML elements can be nested (this means that elements can contain other elements). All HTML documents consist of nested HTML elements.

    Let's take a look at below example:

      <!DOCTYPE html>
  <html>
  <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <title>Document</title>
  </head>
  <body>

      <h1>My first heading</h1>
      <p>My first paragraph</p>

  </body>
  </html>

    This example contains six elements <html>, <head>, <meta>, <body>, <h1> and <p>.

    The <meta> tag is nested inside the <head> element

    The <h1> and <p> elements are nested inside the <body> element

    And remember, most elements needs to have a closing tag

    1. Let us try nesting a link inside a <div> element:

      The <div> tag defines a division or a section in an HTML document and is used as a container for HTML elements. Any sort of content can be put inside this tag. 

        <body>

      <h1>My first heading</h1>
      <p>My first paragraph</p>

      <div>
          <a href="https://www.skaggbyran.se/">My first paragraph</a>
      </div>

  </body>
